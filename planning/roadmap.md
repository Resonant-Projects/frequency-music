# Roadmap & Milestones

## Milestone 0 — Repo + skeleton (1–2 days)
- Astro site scaffold (static pages)
- Solid app scaffold + Clerk auth
- Convex project initialized
- UploadThing configured

## Milestone 1 — Ingestion (1–2 weeks)
- Notion tag-based one-way sync via n8n → Convex
- Manual URL + YouTube paste UI
- PDF upload UI (UploadThing) + store asset URL
- Basic dedupe + “Inbox” view

## Milestone 2 — Extraction (1–2 weeks)
- Convex Action calling LLM for `extract_v1`
- Store extractions; allow manual edits
- Display extracted composition parameters cleanly

## Milestone 3 — Hypotheses/Recipes (1 week)
- Hypothesis CRUD + citations
- Recipe generator + manual edits
- Composition project creation from recipe

## Milestone 4 — Weekly Brief (1 week)
- Weekly scheduled generation
- Editable brief editor
- Publish to followers

## Milestone 5 — Collaborator feedback (2–4 days)
- Listening session logging
- Permissions for collaborator role

## Post-MVP
- Connection suggestions / lightweight knowledge graph
- Better PDF text extraction pipeline
- Public (no-login) mode + SEO
- Measurement modules (later)
